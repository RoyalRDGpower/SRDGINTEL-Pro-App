# SRDGINTEL Pro App

This is a professional AI Video Generation platform built with [Next.js](https://nextjs.org).

## Features

*   **Blog Feed**: Dynamic AI tech news feed.
*   **Video Studio**: High-end gallery showcase.
*   **Dashboard**: Project tracking and analytics.
*   **Video Wizard**: Step-by-step AI video generation UI.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
